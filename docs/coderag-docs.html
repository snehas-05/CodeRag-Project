01 — Project Overview

What is CodeRAG?

CodeRAG is an AI-powered debugging assistant that helps developers analyze, understand, and fix issues in large codebases. It combines Retrieval-Augmented Generation, a LangGraph-based reasoning agent, and locally-hosted AI models to deliver accurate root cause analysis — with zero API dependency.

🧠 Core AI

RAG + LangGraph

Retrieval-powered context feeding into iterative reasoning loops for high-confidence answers.

🔒 Local Models

Fully Offline

CodeBERT, Sentence Transformers, and FLAN-T5 — all cached and run locally inside Docker.

How CodeRAG Solves It

04 — Core AI Concepts

The Intelligence Engine

Two AI paradigms working in tandem: RAG provides the knowledge, LangGraph provides the reasoning.

🔍 Retrieval

Retrieval-Augmented Generation

RAG enhances AI responses by retrieving verified information before generation — eliminating hallucination.

  • Retrieves code snippets, logs, and documentation
  • ChromaDB for semantic vector search
  • Elasticsearch BM25 for log retrieval
  • Answers grounded in real code
  • Dramatically reduces hallucination
🔁 Reasoning

LangGraph Agent

LangGraph enables iterative decision-making loops — the agent keeps reasoning until it's confident.

  • Retrieve Node → fetch relevant context
  • Analyze Node → generate hypothesis
  • Verify Node → validate hypothesis
  • Low confidence → loop back and retry
  • Respond Node → finalize structured answer
💡 Key Insight: RAG provides knowledge. LangGraph provides reasoning. Together, they produce trustworthy, evidence-backed debugging outputs.

06 — End-to-End Workflow

From Repo to Root Cause

Six steps from a GitHub repository to a structured debugging answer.

1

Repository Ingestion

GitHub repo cloned, parsed, AST-based chunking extracts functions and classes.

2

Embedding & Storage

Code converted into vector embeddings and stored in ChromaDB for retrieval.

3

User Query

Developer asks: "Why is my API failing?"

4

RAG Retrieval

Relevant code snippets and logs fetched via vector + BM25 hybrid search.

5

LangGraph Reasoning

Agent generates hypothesis → verifies → loops until high confidence reached.

6

Structured Response

Root cause + evidence (file + line) + suggested fix delivered to the user.
